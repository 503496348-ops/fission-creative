# 裂变创作-Fission Creative

**版本**: v6.0.0 | **协议**: GPL v3 | **作者**: AtomCollide-AI-陈宇锋团队

长篇网文辅助创作系统 — 解决AI写作中的「遗忘」和「幻觉」问题，支持200万字量级连载创作。

基于 lingfengQAQ/webnovel-writer v6.0.0 改造，适配 Hermes Agent 平台。

## 核心能力

| 命令 | 功能 |
|------|------|
| /webnovel-init | 深度初始化 |
| /webnovel-plan | 卷纲规划 |
| /webnovel-write | 章节创作 |
| /webnovel-review | 六维质量审查 |
| /webnovel-query | 状态查询 |
| /webnovel-learn | 经验记录 |
| /webnovel-dashboard | 可视化面板 |

## 安装

```bash
git clone https://github.com/503496348-ops/fission-creative.git ~/.hermes/profiles/<profile>/skills/fission-creative
```
